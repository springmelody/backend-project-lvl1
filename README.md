## Установка игры
1. git clone https://github.com/springmelody/backend-project-lvl1.git
2. make install

## Чтобы поиграть, введите название одной из перечисленных игр
* `brain-even`
* `brain-calc`
* `brain-gcd`
* `brain-progression`
* `brain-prime`
